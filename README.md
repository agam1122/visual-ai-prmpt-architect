# Visual AI Prompt Architect 🎨⚡

**Visual AI Prompt Architect** is a modern, responsive, cyber-dark themed web application designed to help digital artists construct and manage optimized prompt formulas for **Midjourney (V5/V6/V8)** and **Stable Diffusion**. 

Live website: [https://promptarchitects.online](https://promptarchitects.online)

---

## ✨ Features

- **Interactive Tag Selector:** Easily append styles, professional camera lenses, cinematic lighting styles, and curated color palettes using visual chip tags.
- **Advanced Parameters Slider:** Calibrate technical Midjourney settings dynamically (Aspect Ratio `--ar`, Stylize `--s`, Chaos `--c`, Weird `--w`, and Raw Mode `--style raw`).
- **Dynamic Preview & Metrics:** A live prompt compilation output with real-time word and character counters.
- **Local Saved Library:** Save, view, copy, or delete your customized prompt formulas directly within your browser (stored safely via local storage).
- **Responsive Cyber-Dark UI:** Premium glassmorphism design with a mobile-first responsive layout and quick-share shortcuts.

---

## 🛠️ Technology Stack

- **Structure:** HTML5 semantic markup with JSON-LD schema (FAQ and WebApplication schemas).
- **Styling:** Custom CSS3 styling built on native variables for high-performance and lightweight PageSpeed scores.
- **Logic:** Vanilla JavaScript for DOM updates, clipboard copy APIs, and local storage integration.

## 🚀 Getting Started

To run the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/visual-ai-prompt-architect.git
   ```
2. Open the directory:
   ```bash
   cd visual-ai-prompt-architect
   ```
3. Open `index.html` directly in any web browser, or serve it using a local development server.
